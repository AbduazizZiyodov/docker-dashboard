I decided to create a dashboard for working with Docker easily. Now, you can manage your containers and images.
Containers

A Docker container image is a lightweight, standalone, executable package of software that includes everything needed to run an application. In this section, you can manage your containers.

Images

A Docker image is a file used to execute code in a Docker container. Docker images act as a set of instructions to build a Docker container, like a template. In this section, you can manage your images.
